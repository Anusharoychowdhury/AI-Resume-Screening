# 🤖 AI Resume Screening System

## 📌 Overview

This project is an AI-powered resume screening system that ranks resumes based on their similarity to a given job description.

It helps recruiters quickly identify the best candidates using Natural Language Processing (NLP).

---

## 🚀 Features

* Upload multiple resumes (PDF)
* Extract text from resumes
* Clean and preprocess text
* Match resumes with job description
* Rank candidates based on similarity score
* Display top candidates with visualization
* Download results as CSV

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* NLP (TF-IDF / Similarity)

---

## 📂 Project Structure

```
app.py
resume_parser.py
job_matching.py
requirements.txt
```

---

## ▶️ How to Run

1. Clone the repo:

```
git clone https://github.com/Anusharoychowdhury/AI-Resume-Screening.git
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run app.py
```
